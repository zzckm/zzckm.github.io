# 大数据01-集群搭建

## **1. 集群规划**

集群节点分配
: 
    |主机名|主机IP|
|---|---|
|master|192.168.100.100|
|slave1|192.168.100.101|
|slave2|192.168.100.102|
|slave3|192.168.100.103|

软件版本
:   
    |软件名称|版本号|
|---|---|
|Java|1.8.0_172|
|CentOS|CentOS-7-x86_64-DVD-1708|
|Hadoop|2.7.6|

各软件安装路径
:  
     |软件名|路径地址|
|---|---|
|Hadoop|/opt/apps/Hadoop|
|Java|/opt/apps/Java|

## **2. 基础软件服务安装**
2.1 ssh安装
: 安装完整的ssh服务，文件上传以及远程操作都是该服务
    ```shell
    yum -y install openssh-clients
    
    ```
